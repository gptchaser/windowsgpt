# WindowsGPT
一款Windows应用程序，可以通过ChatGPT让你与系统操作进行交互。

##### A windows app that allows you to use ChatGPT to interact with system operations.
### 安装 Install
```
pip install -r requirements.txt
```
### 运行 Run
```
python windowsgpt.py
```
